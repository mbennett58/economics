## 📂 Project Structure

* `/apps`: Custom HTML5/JavaScript interactive tools (simulations, calculators, etc.).
* `/past-papers`: Organized folders of PDF past papers and mark schemes.
* `/assets`: CSS stylesheets, branding images, and shared JavaScript libraries.
* `index.html`: The main landing page for students.

## 🚀 How to Add New Resources

### Adding a Past Paper
1. Navigate to `/past-papers/[Subject]/[Year]`.
2. Upload the PDF file.
3. Update the corresponding HTML page with a link to the new file.

### Adding an Interactive App
1. Create a new folder in `/apps/your-app-name`.
2. Include an `index.html` file within that folder.
3. Students can then access it at `yourusername.github.io/apps/your-app-name`.
